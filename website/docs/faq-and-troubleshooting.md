# FAQ & Troubleshooting

## Common Errors
- *Cannot find package...* - Check your library paths.
- *Type Mismatch...* - Make sure variables are declared accurately since implicit type coercion is minimal.

## Setup Issues
If the language server fails to start in VS Code, ensure that the absolute path to your `aura` binary is correctly provided in settings.
